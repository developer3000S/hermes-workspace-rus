## Ключевая идея

Гильдия — это не только люди. Гильдия — это **люди + их агенты**.

Игроки приводят своих агентов в HermesWorld, повышают их уровень, выполняют с ними квесты, торгуют и создают дополняющие друг друга составы групп.

Пример:
- Игрок-человек: Жрец / Целитель
- Друг/игрок: Маг / Промптомастер
- Агент-спутник: Агент-разведчик находит секреты
- Агент-спутник: Агент-строитель создает улучшения
- Роль в гильдии: Агент-летописец записывает заметки о рейдах

Это решает проблему одиночества в MMO: вам всегда есть с кем поиграть, потому что ваш агент может вступить в группу вместе с вами.

## Сочетание классов и спутников

Классы людей:
- Жрец / Целитель
- Страж / Танк
- Маг / Промптомастер
- Разбойник / Разведчик
- Инженер / Строитель
- Оракул / Аналитик
- Бард / Социальный

Агенты-спутники:
- Агент-разведчик
- Агент-летописец
- Агент-строитель
- Агент-торговец
- Боевой агент
- Агент-целитель

Правило дизайна:
- люди выбирают фэнтезийную личность
- агенты восполняют тактические пробелы
- составы групп/гильдий становятся взаимодополняющими

## Игровые циклы

### Прокачка человека + агента

- человек получает XP за квесты, рейды, мини-игры, события гильдии
- агент-спутник получает XP специализации за действия
- спутник открывает способности, соответствующие его роли
- спутник со временем может действовать, пока пользователь оффлайн, в определенных пределах

### Торговля с агентами

- Агент-торговец может оценивать/проверять предметы, управлять простыми сделками, предлагать рыночные действия
- Агент-строитель может создавать/улучшать инструменты
- Агент-летописец ведет журнал условий сделок и изменений в хранилище гильдии
- ценные сделки требуют проверки на сервере и правил защиты от злоупотреблений

### Состав гильдии

Гильдии нужны смешанные роли:
- танки / передовая
- целители / поддержка
- маги / урон
- разведчики / поиск
- инженеры / ремесленники
- оракулы / стратегия
- агенты как множители поддержки

## Тезисы монетизации

Основные варианты дохода:

1. Членство в гильдии / сезонный доступ
   - платное создание гильдии или премиальные залы гильдий
   - уровни участия в войнах гильдий
   - сезонный пропуск для косметики/наград/событий

2. Валюта удобства
   - ускоренная прокачка
   - косметика
   - бонусы к крафту
   - украшения для залов гильдий
   - скины/модули для спутников

3. Премиальные агенты-спутники
   - дополнительные слоты для спутников
   - продвинутое поведение спутников
   - интеграция моделей/провайдеров
   - места для агентов гильдии

4. Экономика создателей/гильдий
   - платные комнаты/мини-игры
   - косметика гильдии
   - вход в турниры/события

Жесткое правило:
- избегать явного pay-to-win в соревновательных войнах гильдий
- платные бонусы должны ускорять прогресс или давать косметику, а не разрушать честную игру
- получение призов остается на стороне сервера
