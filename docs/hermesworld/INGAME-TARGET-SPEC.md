# HermesWorld — Спецификация внутриигрового целевого вида

Последнее обновление: 2026-05-06 02:22 EDT

Ссылка: `docs/hermesworld/reference-images/INGAME-TARGET-AGORA.png`

Это **игровой вид**, который подразумевается маркетинговой графикой.
Когда пользователь находится *внутри* HermesWorld, его экран должен выглядеть именно так.

---

## Что показано на референсе

Круглая площадь в Agora Commons:
- изометрическая камера / вид из-за плеча (ближе к высокому ракурсу 3/4, чем к виду сверху)
- центральный каменный монумент с золотым пирамидальным навершием, окруженный зажженными факелами
- радиальная плитка из булыжника с руническими кругами
- скамейки, растительность, фонари по краям
- торговые лавки (красные/синие палатки) с ящиками и бочками
- 6 NPC вокруг площади, каждый с цветным именным тегом
- 1 аватар игрока (Eric) в центре сцены с плашкой имени
- огонек/искра справа спереди (подсказка для сбора/квеста)

### Макет HUD
- **Верхний левый угол**: карточка игрока
  - портрет
  - имя "Eric"
  - подзаголовок зоны "TRAINING GROUNDS"
  - XP "75 - next 25"
  - 4 баббла характеристик: HP 100, MP 50, SP 80, XP 75
- **Верхний центр**: баннер цели
  - иконка + заголовок "OBJECTIVE — Move and Speak"
  - подтекст "Walk to Athena and speak with her. Athena waits by the Arrival Circle."
- **Верхний правый угол**: миникарта "AGORA COMMONS"
  - круглая карта
  - маркер ВЫ (YOU)
  - маркер портала
- **Средний левый край**: небольшой эмбиент-стикер "Hermes Guide" (карточка помощника / уведомление от NPC)
- **Правый край**: вертикальная панель иконок (настройки, полноэкранный режим, поделиться, группа и т.д.)
- **Нижний левый угол**: панель чата
  - "CHAT - 4 ONLINE - 3 NPC LOCAL-ONLY"
  - цветные теги NPC у говорящих
  - поле ввода чата с кнопкой ОТПРАВИТЬ (SEND)
- **Нижний центр**: панель способностей
  - тройная полоса HP/MP/SP слева
  - 4 слота способностей с таймерами кулдауна/иконками + 1 модификатор
- **Именные теги NPC**: парят над каждым NPC с указанием роли
  - "Athena - Sage", "Apollo", "Nora - Piper", "Silas - Guard", "Dorian - Quartermaster"
- **Всплывающие окна речи**: эмбиент-фразы над головами NPC
  - "Starter kit, cheap and proud."
  - "gm builders"

### Атмосфера
- теплое освещение "золотого часа"
- премиальный вид 3D-рендеринга (уровень Genshin / Honkai / Dragon Quest XII)
- четкие оверлеи UI со стеклянными панелями и теплым внутренним свечением
- мягкие частицы в воздухе (огоньки, пылинки возле факелов)

---

## Почему это важно

Маркетинговая графика (`MASTER-PRODUCT-GRAPHIC.png`) — это то, как должен выглядеть **сайт или страница в магазине**.

Этот внутриигровой целевой вид (`INGAME-TARGET-AGORA.png`) — это то, как должна выглядеть **сама игра**.

Оба варианта должны соблюдаться. Они должны ощущаться как один и тот же продукт.

---

## Стилевые ограничения (Style locks)

- **Камера**: 3/4 изометрия, небольшой поворот, наклон ~45-50°
- **Освещение**: теплый золотой час с контурным светом (rim light) на персонажах
- **Поверхность**: нарисованные вручную булыжники с инкрустированными руническими кругами
- **Пропсы**: стилизованный камень, дерево, ткань, бронзовые детали
- **Персонажи**: пропорции с уклоном в чиби, плащи, робы, энергетика Dragon Quest x Genshin
- **NPC**: отчетливые силуэты, цветовое кодирование по фракциям/ролям
- **HUD**: темный стеклянный хром, золотые/янтарные акценты, белый текст, небольшой скос (bevel)
- **Именные теги**: в форме пилюли, темные со светлым текстом, суффикс роли
- **Баннер цели**: темная плашка в верхней центральной части, иконка слева, двухстрочный текст
- **Миникарта**: круглая, темная внутри, светящиеся ориентиры, маленькие маркеры
- **Чат**: 5-цветные теги для типа источника (NPC, игрок, система), моноширинный UI

## Список ассетов на основе этого изображения

### 3D / мир
- пол круглой площади (булыжники, рунические круги)
- центральный монумент (каменные ярусы + золотое пирамидальное навершие)
- факелы (низкие + высокие)
- фонари
- скамейки
- ящики и бочки
- торговые лавки (красные, синие, зеленые крыши)
- группы растительности
- плитки грунтовой тропинки на границе с травой
- арка портала (используется в соседнем изображении)
- атмосферные частицы

### Персонажи / агенты
- шаблон аватара игрока с вариантами классов:
  - Priest, Guardian, Mage, Rogue, Engineer, Oracle, Bard
- список NPC (Агора):
  - Athena - Sage
  - Apollo
  - Nora - Piper
  - Silas - Guard
  - Dorian - Quartermaster
  - Hermes Guide (помощник)
- агенты-компаньоны согласно маркетинговой графике

### UI
- панель карточки игрока
- бабблы характеристик (HP/MP/SP/XP)
- баннер цели
- миникарта с маркерами
- правая панель иконок (настройки, полноэкранный режим, поделиться, группа, соцсети)
- панель чата
- панель способностей с рамками слотов + кулдауном
- плашки именных тегов
- бабблы речи
- карточки уведомлений (toast cards)

### FX
- пламя факелов + свечение
- огоньки / искры
- пылинки
- частицы шагов

## Технические цели

- Камера: ортографическое 3D в Three.js / R3F
- Формат ассетов: glTF/GLB с оптимизированными текстурами (KTX2/Basis)
- Шейдеры: стилизованный PBR + контурный свет + дешевый туман
- Производительность: 60fps на M-серии, цель 30fps на ноутбуках среднего сегмента
- Мобильные устройства: упростить панель чата + миникарту, оставить панель способностей

---

## Цикл реализма для этого вида

Для каждой итерации:
1. сделать скриншот текущей играбельной Агоры
2. сравнить с `INGAME-TARGET-AGORA.png`
3. составить список из 3 главных видимых отличий (камера, освещение, монумент, факелы, NPC, HUD, панель чата и т.д.)
4. написать минимальный патч
5. снова сделать скриншот
6. повторить

Так мы плавно переходим от текущего состояния к целевому без переписывания движка.
