## Стиль: Премиальное темное фэнтези с циановыми/янтарными акцентами

**Референс настроения**: соответствует `INGAME-TARGET-AGORA.png` и `MASTER-PRODUCT-GRAPHIC.png`.

### Палитра (заблокирована на основе бренд-листа сообщества от 2026-05-06)

Точные HEX-значения:

- **GOLD** (Золото) `#F1C56D` — основной акцент
- **BRONZE** (Бронза) `#B8862B` — вторичный акцент
- **PARCHMENT** (Пергамент) `#F4E9D3` — светлый текст / бумага
- **VERDIGRIS** (Ярь-медянка) `#2E6A63` — холодный акцент мира
- **MIDNIGHT** (Полночь) `#0F1622` — базовый темный
- **SLATE** (Сланец) `#1B2433` — темный для панелей
- **STONE** (Камень) `#8A8F98` — нейтральный
- **OBSIDIAN** (Обсидиан) `#0A0D12` — глубокий темный

Градиенты:
- **Золотой градиент**: `#F1C56D` -> `#B8862B`
- **Градиент мира**: `#0F1622` -> `#2E6A63`

Избегать: перенасыщенного неона, чистого черного, плоского серого, цветов вне палитры.

### Освещение

- основной свет (key light): теплый "золотой час"
- контурный/заполняющий свет (rim/fill light): циановый/бирюзовый
- мягкая волюметрическая дымка
- видимый блум (bloom) факелов/фонарей
- едва заметные частицы (пылинки, огоньки)

### UI

- темный стеклянный хром поверх Midnight/Slate, полупрозрачный
- золотые (#F1C56D) акцентные обводки
- внутреннее свечение на важных панелях
- текст: пергамент (#F4E9D3) для основного текста, белый для заголовков
- иконки: линейная графика (line-art) золотого цвета (единая толщина линий, см. раздел "Язык иконок")
- именные теги: темная "пилюля", светлый текст, суффикс роли в каменном/приглушенном цвете
- активные состояния: бронзовое подчеркивание + золотое свечение

### Язык иконок (заблокирован на основе бренд-листа сообщества)

Линейные иконки, золотая (#F1C56D) обводка, единая толщина. Канонический набор:
- Sigil (кадуцей в круге)
- Quest (лампочка с пометкой)
- Compass (компас)
- World (геодезический многогранник)
- Portal (арка)
- Inventory (сумка)
- Scroll (свиток)
- Chat (баббл речи)

Новые иконки должны следовать этому же языку линейной золотой графики.

### Типографика (заблокирована)

- **Display serif** (Акцидентный шрифт с засечками): Canela / Instrument Serif (для логотипа HermesWorld, главного текста, названий зон)
- **UI / body** (UI / основной текст): Inter / Söhne (для HUD, кнопок, чата)
- **Начертания**: Light / Regular / Medium / Semibold / Bold

### 3D

- стилизованный PBR
- пропорции персонажей с уклоном в чиби
- выразительные силуэты
- ощущение текстур, расписанных вручную
- 3/4 изометрия, наклон ~45-50°
- цель 60fps на M-серии, 30fps на ноутбуках среднего сегмента
