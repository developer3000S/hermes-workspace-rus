# Статус генерации активов HermesWorld V2

Ветка: `feat/asset-generation-v2`

## Статус

Генерация изображений заблокирована до создания каких-либо изображений, так как инструмент генерации изображений Hermes включен, но учетные данные провайдера не настроены в этом рантайме.

Ошибка инструмента:

```text
functions.image_generate: ValueError: FAL_KEY environment variable not set
```

В этом запуске не было зафиксировано (committed) никаких сгенерированных растровых активов. Это позволяет избежать поставки синтетических заглушек при утверждении, что они получены из imagegen.

## Целевые пути вывода

- Варианты зон и выбранные ключевые изображения зон: `public/assets/hermesworld/zones/v2/`
- Портреты NPC: `public/avatars/v2/`
- Лист спрайтов иконок и манифест: `public/assets/hermesworld/icons/sprite-v1.png` и `sprite-v1.json`
- Постер видео: `public/assets/hermesworld/video/world-demo-poster-v2.jpg`

## Источник промптов

Используйте `docs/hermesworld/PROMPT-LIBRARY.md` для получения точных повторяемых промптов и значений стилевых настроек (style lock).

## Команда для возобновления

После установки `FAL_KEY` в среде выполнения Hermes повторно запустите процесс генерации активов для этой ветки и зафиксируйте (commit) сгенерированные активы.
