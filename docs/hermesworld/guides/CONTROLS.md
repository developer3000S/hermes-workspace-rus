# Управление (Controls)

HermesWorld поддерживает управление с помощью клавиатуры и мыши (для ПК), а также сенсорное управление (для мобильных устройств). Элементы управления должны оставаться интуитивно понятными даже в динамичных ситуациях и никогда не перекрывать основные области интерфейса (HUD).

См. также: [Начало работы](GETTING-STARTED.md), [Социальное взаимодействие](SOCIAL.md).

## Движение на ПК

| Действие                             | Клавиша                                  |
| ------------------------------------ | ---------------------------------------- |
| Движение вперед / влево / назад / вправо | `W` `A` `S` `D`                          |
| Альтернативное движение              | Клавиши со стрелками                     |
| Спринт / ускорение (если доступно)   | `Shift`                                  |
| Взаимодействие / разговор / подтверждение | `E` или экранная подсказка               |
| Отмена / закрыть диалог              | `Esc`                                    |
| Открыть чат                          | `Enter`                                  |
| Отправить сообщение в чат            | `Enter` (при активном поле ввода)        |
| Переключить карту / миникарту        | `M` (если доступно)                      |
| Ячейки способностей                  | Цифровые клавиши `1`–`4` (если доступно) |
| Модификатор / специальное действие   | `Пробел` или кнопка интерфейса           |

## Мышь

| Действие                        | Управление                          |
| ------------------------------- | ----------------------------------- |
| Выбор в меню / нажатие кнопки   | Левая кнопка мыши                   |
| Вращение камеры (если доступно) | Правая кнопка мыши или перетаскивание |
| Просмотр подсказок (тултипов)   | Наведение курсора                   |
| Прокрутка панелей               | Колесико мыши / тачпад              |

## Мобильные устройства

| Действие                        | Управление                               |
| ------------------------------- | ---------------------------------------- |
| Движение                        | Левый виртуальный джойстик               |
| Взаимодействие / основное действие | Кнопка действия на правой панели         |
| Вторичные действия              | Дополнительные кнопки на правой панели   |
| Чат                             | Панель чата / кнопка ввода               |
| Камера / обзор                  | Перетаскивание по свободной области      |
| Закрыть диалог                  | Кнопка закрытия или нажатие вне панели   |

## Правила расположения элементов на мобильных устройствах

Мобильный интерфейс должен сохранять три «священные зоны»:

1. **Зона джойстика (слева внизу)** — никогда не перекрывайте её диалогами, уведомлениями или облачками речи.
2. **Зона действий (справа)** — кнопки взаимодействия и боя должны быть легко доступны.
3. **Верхняя зона (цели и миникарта)** — компактный статус без ущерба для обзора движения.

Облачка речи должны парить над головами и уменьшаться на мобильных устройствах. Уведомления (тосты) должны располагаться высоко по центру, вдали от зон джойстика и кнопок действий.

## Управление диалогами

Диалоги с NPC отображаются в виде «пергаментных» облачков речи. Варианты выбора могут появляться в виде кнопок под репликой NPC.

- Используйте клик или касание для выбора.
- Используйте `Esc` или кнопку закрытия для выхода.
- Диалоги в стиле чата сохраняют историю последних реплик.

## Примечания по доступности

Запланированные настройки:

- Уменьшение интенсивности движения.
- Увеличенный шрифт.
- Высококонтрастный интерфейс.
- Возможность переназначения клавиш на ПК.
- Настройка размера джойстика.
- Регулировка прозрачности чата.

Пока эти функции находятся в разработке, основное правило дизайна остается простым: если кнопка выглядит красиво, но на нее нельзя нажать — это картина, а не интерфейс.
