# Арт-библия HermesWorld — Цикл реализма

Последнее обновление: 06.05.2026

## Визуальные амбиции

HermesWorld должна ощущаться как качественная браузерная RPG в стиле фэнтези/сай-фай с участием агентов, а не как демонстрационная сцена.

Целевая атмосфера:
- кинематографичное темное фэнтези в сочетании с командным центром агентов
- гибрид реализма и стилизации, хорошо читаемый в масштабе браузера
- атмосферное освещение, туман, излучающий свет магический интерфейс, высокий контраст
- интерфейс, ориентированный в первую очередь на игру, а не на загроможденную панель SaaS
- каждый актив должен поддерживать игровые механики

Основная цель для Агоры:
- Используйте `docs/hermesworld/reference-images/agora-center-inso-reference.jpeg` в качестве основного визуального ориентира для работы над реализмом Агоры.
- Цель — скомпонованный игровой экран: круглая городская площадь, центральный обелиск/монумент, теплый свет факелов, синие/голубые акценты на крышах/порталах, плотные края рынка, свободный центр для прогулок, читаемые группы NPC и компактные островки HUD.
- Промпты для активов и манифест находятся в `docs/hermesworld/AGORA-INSO-ASSET-PROMPTS.md`.

Справочный рабочий процесс из загрузок Эрика:
- Генерация изображений (imagegen) создает высококачественный исходный материал.
- Игра размещает эти активы в реальном интерфейсе.
- Скриншоты браузера фиксируют фактический вид игрока.
- Визуальный обзор (Vision review) оценивает иерархию, отступы, читаемость, удобство кликов, соответствие мобильным устройствам и визуальную согласованность.
- Агенты вносят правки и повторяют цикл.

## Палитра

Базовая:
- Обсидиан: `#080F14`
- Панель: `#18212B`
- Глубокие чернила: `#030712`

Акценты:
- Активный зеленый: `#2FCA94`
- Визуальный синий: `#78A8C8`
- Интуитивный янтарный: `#F2C768`
- Тайный пурпурный: `#8B5CF6`, только для контролируемой магии/свечения

Текст:
- Основной: `#E6E7EA`
- Приглушенный: `rgba(230,231,234,.65)`

Избегать:
- перенасыщенного неона повсюду
- пурпурного тумана как единственного настроения
- стиля милых/мультяшных маскотов
- нечитаемого крошечного текста
- плотности дашборда SaaS на мобильных устройствах

## Язык форм

Панели:
- радиус скругления 6-10 пикселей
- низкоконтрастные рамки в 1 пиксель
- тонкие внутренние блики
- материалы типа темного стекла/металла

HUD:
- компактные островки, а не цельные плиты
- состояние видно с первого взгляда
- иконки читаемы при исходном размере 48-64 пикселя
- для мобильных устройств используется своя раскладка, а не сжатая десктопная

Мир:
- в первую очередь крупные силуэты
- читаемые ориентиры
- один основной источник света на сцену
- вторичные огни от костров/магии/порталов
- атмосферная глубина, но не скрывающая навигацию

## Правила создания активов

Генерируйте с помощью imagegen:
- ключевой арт / баннеры зон, 16:9
- портреты NPC, квадратные 1:1, согласованное кадрирование
- иконки предметов, квадратные 1:1, читаемые при 64 пикселях
- иконки сигилов, квадратные 1:1, высокий контраст
- карточный арт, вертикальный 2:3
- фоны окружения, 16:9
- исследование текстур/материалов

Не генерируйте финальную раскладку с помощью imagegen.
Раскладка должна оцениваться по скриншотам браузера.

Стиль: ключевой арт игры класса AAA, палитра темного обсидиана, голубой/янтарный акцентный свет, тонкое тайное свечение. 16:9.

## Чек-лист визуального обзора

Каждый скриншот должен оцениваться по следующим критериям:
1. Игровой процесс прежде всего: может ли игрок понять, что делать?
2. Видимость состояния: HP/цель/карта читаются мгновенно?
3. Читаемость HUD: текст, иконки и области нажатия четкие?
4. Плотность инвентаря: предметы различимы?
5. Мобильная раскладка: без прокрутки, без наложений, без сжатого десктопного UI.
6. Арт поддерживает механику: ориентиры, NPC, объекты передают свое назначение.
7. Визуальная согласованность: та же палитра, логика света, кадрирование, рамки.
8. Производительность: активы оптимизированы, а не просто красивы.

## Цели реализма HermesWorld v0.3

Минимальный видимый скачок:
- замена плоских/заглушечных изображений зон на сгенерированные кинематографичные баннеры
- 5 портретов NPC с согласованным премиальным кадрированием
- 12 иконок предметов с читаемыми силуэтами
- 7 иконок сигилов для слоя лора/пасхалок
- HUD переработан в компактные островки игрового стиля
- по одному высококачественному циклу скриншотов для мобильных и десктопных устройств

## Цикл сборки

1. Сгенерировать набор активов.
2. Сжать/оптимизировать в веб-активы.
3. Разместить в фактическом интерфейсе HermesWorld.
4. Сделать скриншот.
5. Провести визуальный обзор по чек-листу.
6. Исправить UI/арт/раскладку.
7. Повторять, пока это не станет похоже на игру, а не на прототип.
