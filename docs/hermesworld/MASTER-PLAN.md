## 1. Идентичность

HermesWorld = первая **Агентная MMO (Agentic MMO)**.

- Roblox + WoW + Rohan + ИИ-агенты
- Люди + их ИИ-агенты являются согильдийцами
- Сначала браузерная (browser-native), скачиваемая версия позже
- Создается одним основателем + роем агентов
- Нарратив для поглощения (Acquisition narrative): первая компания по разработке видеоигр с ИИ-агентами

Два заблокированных референса:
- **Спецификация для маркетинга/страницы в магазине**: `docs/hermesworld/reference-images/MASTER-PRODUCT-GRAPHIC.png`
- **Спецификация играбельного внутриигрового вида**: `docs/hermesworld/reference-images/INGAME-TARGET-AGORA.png`

Маркетинговая графика — это то, как выглядит сайт. Внутриигровой целевой вид — это то, как выглядит сама игра во время игры. Оба варианта должны соблюдаться. См. `INGAME-TARGET-SPEC.md` для полной разбивки HUD, макета, NPC, пропсов, освещения и привязок к этапам (milestones).

## 2. Столпы видения (из мастер-графики)

- **Выберите свой класс (Choose Your Class)** — 7 человеческих классов
  - Жрец (Priest) / Лекарь (Healer)
  - Страж (Guardian) / Танк
  - Маг (Mage) / Промпткастер (Promptcaster)
  - Разбойник (Rogue) / Разведчик (Scout)
  - Инженер (Engineer) / Строитель (Builder)
  - Оракул (Oracle) / Аналитик
  - Бард (Bard) / Социальный класс

- **ИИ-агенты компаньоны (AI Agent Companions)** — 6 классов агентов
  - Разведчик, Писец, Строитель, Торговец, Боец, Лекарь
  - Слоган: "Ваша Гильдия. Ваши Агенты. Ваше Преимущество."

- **Системы гильдий**
  - Залы гильдий (строительство и кастомизация)
  - Знамена
  - Чат гильдии
  - Цели гильдии
  - Общее хранилище
  - Войны выходного дня (Weekend Wars)
  - Рейды
  - Таблицы лидеров
  - Сезонные награды

- **Эпические события**
  - Захват обелисков / сигилов (контроль карты)
  - Защита зала гильдии
  - Боксерская арена
  - новые события добавляются каждый сезон

- **Имиджевые слоганы**
  - "Ваш Мир. Ваша Гильдия. Ваше Наследие."
  - "Один Мир. Множество Легенд. Бесконечные Приключения."

- **Столпы платформы**
  - Браузерная (не требует скачивания)
  - Мультиплеер в реальном времени
  - Экономика, управляемая игроками
  - Децентрализованное владение
  - Сезонный пропуск (Сезон 1: Рассвет Легенд / Dawn of Legends)

## 5. Волны замены ассетов (правильный порядок)

Мы заменяем арт в HermesWorld партиями (batch by batch), никогда все сразу.

- **Волна A — Идентичность / HUD** (нет риска для игровой логики)
  - полировка логотипа, иконки карты мира, маркеры миникарты, набор HUD (HP/MP/SP/XP/говорить/бежать/меню/карта), плашка цели, рамки уведомлений (toast frames), бабблы чата, карточки наград за квесты

- **Волна B — Арт персонажей** (высокое визуальное влияние)
  - 7 портретов классов
  - 6 портретов агентов-компаньонов
  - портреты ключевых NPC (Athena, Apollo, Hephaestus и т.д.)
  - исправление кастомизатора аватара (баг с перекрытием лица)
  - миниатюры пресетов

- **Волна C — Карта / арт мира**
  - иллюстрированный фон карты мира
  - 6 главных баннеров зон (Training, Forge, Agora, Grove, Oracle, Arena)
  - карточки разблокировки зон
  - иконографика быстрого перемещения
  - стилизация миникарты

- **Волна D — Предметы / сигилы / лут**
  - 7 сигилов
  - базовые иконки предметов (оружие/броня/реликвии/модули компаньонов)
  - система рамок редкости
  - набор знамени гильдии

- **Волна E — Внутриигровое 3D**
  - HY-World 2.0, Hunyuan, Meshy, Tripo
  - пропсы площади Агора (монумент, факелы, лавки)
  - glTF/GLB → оптимизация → добавление в R3F

Всегда используйте цикл реализма:
промпт → генерация → визуальное ревью → оптимизация → добавление в игру → скриншот → визуальное ревью → патч → повтор.

- Публичная сторона: форма `hermes-world.ai/feedback`

- Ветки бота Discord → задачи с меткой `community-report`
- Упоминания/личные сообщения в X (Twitter) просматриваются и сортируются ежедневно
- Публичная дорожная карта: публикуется на основе выбранных задач из репозитория с меткой `publish:roadmap`

## 10. Чего никогда не должно происходить

- никогда: секреты призов/логика оракула на стороне клиента
- никогда: pay-to-win (плати чтобы победить) в соревновательных войнах гильдий
- никогда: полная замена всего арта сразу без цикла реализма
- никогда: выпуск в релиз без проверки по спецификациям (spec gate)
- никогда: потеря этого документа
