# Hermes Workspace v2.1.0 — Swarm (Рой)

Hermes Workspace 2.1.0 представляет **Swarm** — встроенный интерфейс оркестрации мультиагентов для запуска основного агента Hermes с постоянными рабочими агентами (воркерами).

## Основные моменты

- **Режим Swarm (Рой)**
  - маршрутизация работы от основного агента Hermes в живой рой воркеров
  - постоянные воркеры на базе tmux
  - поверхности диспетчеризации и оркестрации с учетом ролей

- **Доска, отчеты и поток входящих**
  - Поддержка доски / канбана для работы роя
  - отчеты и маршрутизация контрольных точек
  - обработка в стиле входящих сообщений для заблокированных и готовых к проверке задач

- **Маршрутизация оркестратора**
  - контрольные точки воркеров сначала проходят через оркестратора
  - более чистый поток рецензента и лучший контроль над эскалацией задач

- **Улучшения надежности**
  - длительные потоки чата SSE выдерживают периоды фоновой обработки агента
  - исправлено подключение баннера одобрения, теперь запросы на использование инструментов снова видны
  - исправлено удаление только локальных переносимых сессий
  - добавлен резервный механизм дашборда для потоков создания/обновления/форка сессий
  - сохранены патчи надежности рабочего пространства

- **Исправления путей Claude и окружения**
  - каноническая обработка корня Claude
  - улучшенная обработка home/env для профилей и хранилища выполнения

- **Документация и безопасность**
  - добавлена документация по Swarm
  - уточнены инструкции по запасному варианту Docker Skills Hub
  - обновлен путь раскрытия информации SECURITY

## Рекомендуемое краткое описание релиза

Hermes Workspace 2.1.0 поставляет Swarm: встроенную поверхность управления мультиагентами для постоянных рабочих агентов, маршрутизацию с приоритетом оркестратора, потоки Доски + отчетов + входящих, а также набор исправлений надежности для потоковой передачи чата, одобрений, сессий и обработки путей Claude.

## Рекомендуемый пост о запуске

Вышел Hermes Workspace 2.1.0.

Он включает **Swarm**: встроенное мультиагентное рабочее пространство, где один агент Hermes может оркестровать работу постоянных воркеров с живыми контрольными точками, Доской, потоком входящих/проверки и улучшенной маршрутизацией между оркестратором и воркерами.

Также в версии 2.1.0:
- повышенная надежность длительных SSE-потоков чата
- исправление баннера одобрения
- исправление удаления переносимых сессий
- резервный путь через дашборд для действий с сессиями
- улучшения в обработке путей Claude

Если вам нужен контроль над несколькими агентами, не покидая рабочего пространства, — это релиз для вас.
