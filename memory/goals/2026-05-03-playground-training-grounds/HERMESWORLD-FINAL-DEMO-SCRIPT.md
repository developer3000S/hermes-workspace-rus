## Подготовка (5 мин)

1. Жесткая перезагрузка обеих вкладок (`Cmd+Shift+R`), чтобы загрузился последний бандл.
2. Окно 1280×720 или 1920×1080 — закройте DevTools.
3. Системные уведомления отключены, включен режим "Не беспокоить".
4. Сброс прогресса (DevTools → Application → Local Storage → удалите `playerProfile`, затем обновите страницу) для чистого прохождения обучения.
5. Откройте экран Карты один раз перед записью, чтобы подгрузить ассеты.
6. Переключатель звука должен быть **ВЫКЛ** (Mute off) — озвучка воспроизводится автоматически для каждого мира.
7. Проверьте многопользовательский хаб:
   ```bash
   curl https://hermes-playground-ws.myaurora-agi.workers.dev/health
   # {"ok":true,"online":N,"ts":...}
   ```

## План тестирования мультиплеера (перепроверить перед записью)

1. Откройте вкладку A: http://localhost:3005/playground (или 3006).
2. Откройте вкладку B: http://localhost:3006/playground (другой источник = чистая изоляция).
3. Заведите обоих в Training Grounds.
4. Индикатор чата в обеих вкладках: **зеленая точка · 2 players · live**.
5. Напишите "hi" во вкладке A → облачко над головой в A, сообщение в панели чата A, облачко над тем же игроком в представлении B, сообщение в панели чата B.
6. Переведите вкладку A в фоновый режим на 60 секунд. Представление игрока A во вкладке B должно сохраняться.
7. Полностью закройте вкладку A. Вкладка B должна удалить аватар в течение ~1с (событие sendBeacon leave).

## Последняя карточка устранения неполадок

| Если… | Тогда… |
|---|---|
| Индикатор чата говорит LOCAL-ONLY | Проверьте вкладку Network: успешны ли запросы POST /presence на URL workers.dev? |
| Аватар игрока исчезает | Проверьте `__hermesPlaygroundLiveCount` в консоли — очистка на сервере теперь 12с, API гибернации выживает при фоновых вкладках. Должно быть стабильно. |
| Озвучка молчит | Сначала кликните в любом месте (браузеры блокируют звук). Проверьте, что кнопка 🗣️ не в режиме mute. |
| Квест застрял | `localStorage.removeItem('playerProfile'); location.reload()` |
| Forge Gate заблокирован | Сначала завершите все 5 тренировочных квестов. |
| WS хаб упал | `curl https://hermes-playground-ws.myaurora-agi.workers.dev/stats` |
