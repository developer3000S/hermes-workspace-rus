# Итерация 003

## Что реализовано

- Добавлен полный цикл завершения обучения: действие по сбросу и повторному прохождению обучения, более четкое модальное окно празднования завершения, обратная связь по разблокировке Forge Gate, оверлей генерации при первом входе в Кузницу и сохранение состояния доступа к интро Кузницы.
- Улучшена ясность онбординга и игровые ощущения: персонализированное приветствие в заголовке, стрелки-указатели целей, пульсация виньетки при низком HP + предупреждающий тон, а также адаптация меню для мобильных/узких экранов без нарушения десктопной разметки.
- Улучшен UX прогрессии: трекер квестов теперь показывает иконки следующих шагов, журнал стал двухколонным браузером глав с более четкими состояниями активных/завершенных задач, а ближайших удаленных разработчиков ("builders") можно "пингануть" из HUD.
- Обновлена документация для подачи проекта в соответствии с процессом онбординга в Training Grounds, финалом в Кузнице, аудио и историей многопользовательского демо.
