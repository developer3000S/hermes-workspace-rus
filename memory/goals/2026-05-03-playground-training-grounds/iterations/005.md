# Итерация 005 — Обзор камеры мышью + доработка обучения в квестах

## Что реализовано
- Добавлена возможность вращения камеры в 3D-сцене при перетаскивании правой или средней кнопкой мыши.
  - Горизонтальное перетаскивание вращает обзор аналогично стрелкам камеры.
  - Вертикальное перетаскивание настраивает наклон (pitch).
  - Колесико мыши изменяет дистанцию камеры (зум).
  - Контекстное меню блокируется при использовании обзора правой кнопкой мыши.
- Обновлена справка по управлению в боковой панели:
  - Перетаскивание правой/средней кнопкой для вращения камеры.
  - Колесико для зума.
  - Клавиши со стрелками по-прежнему вращают обзор.
- В определения квестов добавлены расширенные обучающие метаданные об Агенте Гермесе:
  - `lesson` (урок) объясняет концепцию Агента Гермеса, которой обучает каждый квест.
  - `payoff` (выгода) объясняет, почему это должно быть важно игроку.
- Улучшены карточки квестов в журнале, теперь они показывают:
  - Блок "Урок Гермеса" (Hermes lesson).
  - Блок "Почему это важно" (Why it matters).
  - Более четкий индикатор состояния квеста: завершен / выбран автоматически / разблокируется после предыдущего квеста.

## Почему
Эрик хотел, чтобы запись видео на десктопе выглядела лучше, и запросил управление камерой мышью аналогично клавишам со стрелками. Он также хотел, чтобы каждый квест было проще просматривать/выбирать/завершать, и чтобы они больше обучали Агенту Гермесу. Этот этап напрямую улучшает управление для записи и превращает журнал в более эффективный инструмент документации и онбординга внутри игры.
